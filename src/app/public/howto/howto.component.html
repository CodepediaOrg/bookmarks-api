Save bookmarks

There are several ways to save bookmarks:

  • The easiest is to use the provided bookmarklets. Bookmarklets are little javascript links that live in the bookmarks toolbar of your browser. Just drag one of the links bellow to your browser toolbar:
    • Save URL to bookmarks.dev - popup - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
    • Save URL to bookmarks.dev - go back - when clicked it redirects you to the New Bookmark form. Once added you go back to the initial page [recommended for phones or smaller resolution devices]

    You can select the text on page that you want to be added to the description of the bookmark before you click the bookmarklet's link

  • Save directly using the button from the navigation menu
  • Use the Save URL to bookmarks.dev chrome extension if you use Chromium based browsers

Search bookmarks

You can search bookmarks by placing your search terms in the Search bar. You can choose to search in Public Bookmarks or in Personal & Favorites, by selecting one of the options beneath the Search Bar. Tags must be embedded in square bracktes [].


Let's say you want to look for personal bookmarks that contain the text exception and are tagged with java. Then would put exception [java] in the search bar, and use the Personal & Favorites option

You have the possibility to save this search by clicking the Save search button. This way it will be displayed in the auto completion when you input text in the Search bar

Search via query parameters

You can execute the same search via query parameters

You can trigger the same search as above by calling the following URL: 
      https://www.bookmarks.dev/search?q=exception [java]&sd=personal
  • q - here is your input search text
  • sd - search domain with two options
    • personal - personal and favorite bookmarks
    • public - public bookmarks (may include personal bookmarks if they come from same user)

This is very handy if you use Web Search tools where you can define custom queries, like Alfred Web Search if you are on Mac
