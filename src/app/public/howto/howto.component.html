Saving bookmarks

There are several ways to save bookmarks:

  • The easiest is to use the Add to bookmarks.dev bookmarklet. Drag and drop this link that into your browser toolbar. When you surf and find a page you want bookmarked just click this link and you will be redirected to the Add new bookmark page. Once added, you will be redirected back to the visited page
  • Save directly using the button from the navigation menu
  • Use the Save URL to bookmarks.dev chrome extension if you use Chromium based browsers
