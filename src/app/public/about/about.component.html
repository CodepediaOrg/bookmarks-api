Bookmarking for Developers & Co

Bookmarks.dev is a bookmark manager created and designed primarily for developers and people who work in the information industry.

To make bookmarking easier it features:

    • Save URL to bookmarks.dev - popup - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
    • Save URL to bookmarks.dev - redirect - when clicked it redirects you to the New Bookmark form. Once added you come back to the initial page [recommended for phones or smaller resolution devices]
  • Use the Save URL to bookmarks.dev chrome extension if you use Chromium based browsers
  • Read Later - list of bookmarks to read later
  • Pinned - pin the bookmarks you intensively use currently
  • Favorites - add public bookmarks to favorites to find them in your personal searches
  • Save searches to use later for easier pin pointing known bookmarks
The best rated public bookmarks are published regularly on Github at https://github.com/CodepediaOrg/bookmarks
